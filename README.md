# BAGUSGAMES - Game Tembak-Tembakan 🔫

Selamat datang di **Game Tembak-Tembakan** yang seru dan interaktif!

## 🎮 Cara Bermain

### Kontrol
- **🖱️ Gerakan Mouse**: Gerakkan arah tembakan (lihat crosshair kuning)
- **🔫 Klik Mouse**: Tembak ke arah musuh

### Tujuan Permainan
- Tembak dan eliminasi semua musuh merah yang jatuh dari atas
- Hindari musuh agar tidak menyentuhmu
- Raih skor setinggi mungkin sebelum health habis

### Mekanik Game
- **Health**: Dimulai dengan 100 HP. Berkurang jika:
  - Tertabrak musuh (-10 HP)
  - Musuh lolos ke bawah (-5 HP)
- **Score**: Dapat poin untuk setiap musuh yang ditembak (10 + Level × 5)
- **Level**: Naik otomatis setiap 500 poin
- **Difficulty**: Semakin tinggi level, semakin banyak dan cepat musuh spawn

## 📊 Fitur Game
✨ Graphics yang colorful dan eye-catching
✨ Sistem level progresif
✨ Particle effects untuk feedback visual
✨ Smooth gameplay dengan 60 FPS
✨ Health & Score tracking real-time
✨ Crosshair aiming system

## 🚀 Cara Menjalankan

1. Buka file `game.html` di browser Anda
2. Game akan langsung dimulai
3. Mulai tembak musuh!
4. Saat Game Over, klik "Play Again" untuk bermain ulang

## 🎯 Tips & Trik
- Jangan fokus pada satu musuh saja, perhatikan yang lain
- Bergerak cepat untuk menghindari musuh yang datang dari segala arah
- Manfaatkan space yang ada untuk menjauh dari musuh
- Semakin tinggi level, semakin banyak poin yang bisa didapat
